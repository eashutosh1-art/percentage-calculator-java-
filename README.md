# 📊 Percentage Calculator (Java)

This is a Java-based application developed using IntelliJ IDEA to calculate Class 10/12 percentage based on subject-wise marks.

---

## 🚀 Features
- Calculates percentage for multiple subjects
- Simple and clean output
- Beginner-friendly Java logic

---

## 🛠️ Technologies Used
- Java
- IntelliJ IDEA

---

## ▶️ How to Run
1. Open project in IntelliJ IDEA  
2. Run the main Java file  
3. Enter marks  
4. Get percentage
